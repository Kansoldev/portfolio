Who am I?

I am Yahaya Oyinkansola, a front end developer building and contributing to products people use that adds some sort of value to their lives. What I Do

WEB DESIGN

Web design is an important aspect in building a website as it focuses on the experience that users get when using your website with some factors involved such as appearance, content, combinations of colours, simplicity and efficiency in other to produce an aesthetically pleasing UI.

Web Development

Creating a website involves the process of taking a design created by a web designer and transforming the UI into a working product that people can use online and depending on the complexity, it can either be a static website that just shows content or a web application that users interact with.

Web Application

A web application is a software program that users interact with on a browser from their devices to perform certain actions such as adding an item to a cart, signing up for an account etc. It is very effective for any business or client looking to gain customers.
